④ 엑셀 시트 작성 가이드

모든 시트(삭제 시트 제외)는 맨 앞에 상품ID 칸이 있습니다. 같은 상품ID를 여러 행에 반복해서 적으면 한 상품 안의 여러 그룹/품목/사이즈로 합쳐집니다.

그룹 시트: 상품ID / 그룹ID(비워두면 자동 부여) / 그룹제목

품목 시트: 상품ID / 그룹ID(또는 그룹제목) / 품목명 / 가로mm / 세로mm / 가격 / 종이무게 / 배송일 / 마진율% / 추가마진율% / 클릭URL / 이미지URL / 다음페이지이동URL / 최우선다음페이지이동URL / 서브옵션(이름:추가금액을 세미콜론으로 연결, 예: 유광골드:2000; 무광레드:1500)

매수별배수 시트: 상품ID / 매수 / 배수

비규격설정 시트: 상품ID / 항목(베이스가격·배수·가로최소·가로최대·세로최소·세로최대) / 값

규격사이즈목록 시트: 상품ID / 가로mm / 세로mm (계산기 화면의 "규격사이즈" 드롭다운에 나오는 목록입니다. 사이즈 하나당 한 행씩 적으세요)

삭제 시트: 상품ID 한 칸만 있습니다. 여기 적힌 상품ID는 Firestore 문서 자체가 통째로 삭제됩니다.

상품ID나 품목명이 비어있는 행은 자동으로 무시됩니다. 새 상품ID를 적으면 새 문서가 만들어지고, 기존 상품ID를 적으면 그 문서 전체가 엑셀 내용으로 덮어써집니다.
